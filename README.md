# figures

Các hình vẽ cho quyển sách [https://dunglq2000.github.io/](https://dunglq2000.github.io/) cũng như các tài liệu khác do mình viết trong tương lai.

Các hình vẽ sử dụng TikZ, được vẽ bởi mình hoặc tham khảo từ các nguồn chính sau.

[1] Jérémy Jean. TikZ for Cryptographers. URL: https://www.iacr.org/authors/tikz/, 2016.

[2] Viguier Benoit. URL: https://gitlab.insa-rennes.fr/bviguier/mri.

[3] Các giải pháp trên https://math.stackexchange.com/.

Mọi đóng góp đều được hoan nghênh. Các bạn có thể mở issue trên repo này hoặc liên lạc với mình qua địa chỉ mail dunglq@yandex.ru.
